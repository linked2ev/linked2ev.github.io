# 01. Create React App 

## 01-1. 환경설치

1) node 설치
  - npm 자동설치 (npm 설치를 하면 npx도 설치됨)

2) 리액트 앱 생성

```
$ npx create-react-app react-test-app
$ cd react-test-app
```

3) 앱 구동

```
$ npm start
```
